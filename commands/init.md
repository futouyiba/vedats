# /vscode-ui-test-init

- [init.zh-CN.md](init.zh-CN.md)

Scaffold Workbench UI tests for a VS Code extension repo.

Steps:
1. Run `npx create-vedats@latest` in the repo root.
2. Verify `tests/ui/**`, `.github/workflows/ui-tests.yml`, and scripts exist.
3. Run `npm run test:ui` and report the artifacts path.
