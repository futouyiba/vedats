# VEDATS

VEDATS = VS Code Extension Debugging and Testing Skills。

本仓库提供 Claude Code 技能工作流（VEDATS）以及可复用的 CLI + Harness，用于为 VS Code 扩展添加 Workbench UI 自动化测试。

## 包

- `vedats-harness`：共享 DSL + artifacts + 失败诊断
- `create-vedats`：脚手架 CLI

## 技能

Claude Code 技能位于 `skills/vscode-ui-test/`。

## 安装技能（Claude Code）

```bash
cc --plugin-dir /path/to/vedats
```
